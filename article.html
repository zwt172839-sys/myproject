CSS实现圣杯布局和双飞翼布局的详细教程

作者：前端开发笔记 发布时间：2025-01-15 阅读量：2580 点赞：128

### 一、圣杯布局介绍

圣杯布局（Holy Grail Layout）是一种经典的三列布局，特点是：

  • 左右两列宽度固定，中间列宽度自适应
  • 中间列优先加载（DOM结构中中间列在前）
  • 三列等高（可选）

### 二、圣杯布局实现步骤

1. 搭建基本HTML结构：

<div class="container">
    <div class="middle">中间列</div>
    <div class="left">左列</div>
    <div class="right">右列</div>
</div>

2. 核心CSS代码：

.container {
    padding: 0 200px; /* 为左右列预留空间 */
    overflow: hidden;
}
.middle {
    float: left;
    width: 100%;
    background: #fff;
}
.left {
    float: left;
    width: 200px;
    margin-left: -100%;
    position: relative;
    left: -200px;
    background: #eee;
}
.right {
    float: left;
    width: 200px;
    margin-left: -200px;
    position: relative;
    right: -200px;
    background: #eee;
}

### 三、双飞翼布局

双飞翼布局是对圣杯布局的优化，解决了圣杯布局中相对定位的问题，核心是给中间列增加一个内层容器：

<div class="container">
    <div class="middle">
        <div class="middle-inner">中间列</div>
    </div>
    <div class="left">左列</div>
    <div class="right">右列</div>
</div>

### 四、总结

圣杯布局和双飞翼布局都是实现三列自适应布局的优秀方案，双飞翼布局无需使用相对定位，兼容性更优，推荐在实际开发中使用。
